создать визуальный интерфейс который работает с аудиофайлами
содержание:
1. есть список альбомов, который на графическом интерфейсе отображается как кнопки,
а последняя кнопка содержит "+"
2. при нажатии на "+" открывается форма ввода названия нового альбома и кнопки создать
3. при выборе конкретного альбома мы переходим к его содержимому:
 - аудиофайлам и исполнителю/лям для него
 - сверху отображается кнопка "назад" и название выбранного альбома
 - после списка файлов есть кнопка "загрузить"
 - при нажатии на "загрузить" добавляются файлы формата mp3 и необходимо указать исполнителя/лей 
 - для каждого файла есть возможность удалить (вместе с связанными исполнителями, базу не захламлять)

4. * прикрутить аудиоплейер для проигрывания аудиофайлов

реализация: 
1. аудиофайлы хранятся в отдельной папке внутри проекта
2. путь до файлов хранится в базе
3. при загрузке файла генерировать уникальное имя
4. база должна быть спроектирована по принципам нормализации
5. применить все 3 паттерна (строитель, наблюдатель, декоратор)
6. сделать всё через гит
